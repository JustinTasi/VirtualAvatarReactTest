# 啟動

- npm install 下載套件
- npm run dev 開啟server

# 使用套件

- ESLint
    - npm run dev => 開啟會檢查格式，不符合格式會無法打開
    - vscode: 可以下載ESLint套件，格式不符會有紅底線
    - webstorm: 手動配置ESLint package至node modules裡(Manual ESLint configuration)
